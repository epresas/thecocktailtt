What are Nuclides?
  • Mixins
    • Style Mixins
  • Partials
    • Grid
  • Colors

    Nuclides are parts of the stylesheet source that are not directly useable on their own. This could be mixins, variables like colors and sizes, or settings.

    To mark a variable or mixin as a nuclide, annotate it with the @nuclide tag, followed by the name of the nuclide.

    Special forms are @color and @mixin.

    Mixins
    Style Mixins
    create-mq($breakpoint, $min-or-max)./src/scss/mixins/_mixins.scss
    Show example
    Creates a media query should a breakpoint is given. Can be useful with mobile-first or desktop-first.
    Parameter
    Optional
    Description
    $breakpoint
    Value in px of the breakpoint to create the media query
    $min-or-max
    Determines if the media query is mobile-first (min-width) or desktop-first(max-width)
    @include create-mq(768, 'min'); @include create-mq($bp-md, 'max');
